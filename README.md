# Нагату

**Аркадная экшен-головоломка с открытым исходным кодом**

«Нагату» — это динамическая кроссплатформенная игра в жанре зуманоид, разработанная на языке Java с использованием фреймворка libGDX. Проект представляет собой полностью открытую (Open-Source) реализацию классической механики уничтожения движущихся цепочек. Игрок управляет центральным орудием, рассчитывая траектории выстрелов и создавая цветовые комбинации, чтобы сдержать наступление змейки и не допустить её проникновения в финальную точку уровня.

# Особенности игры:

- Полная открытость кода: Архитектура проекта спроектирована по принципу Open-Source, что позволяет легко модифицировать уровни, добавлять новые типы снарядов и интегрировать сторонние ассеты.
- Современный стек: Полная оптимизация под мобильную платформу Android 16 (API 36) с поддержкой плавного полноэкранного сглаживания графики (4x MSAA).
- Динамическое замедление (Clutch-механика): Скорость движения цепочки нелинейно падает при критическом приближении к финишу (начиная с 70% пройденного пути), что создает напряженный игровой процесс и дает игроку шанс на камбэк.
- Система цепных комбо (Chain Reactions): При уничтожении группы шаров механика «магнитного притяжения» автоматически смыкает разорванные части поезда. Если на стыке вновь образуется группа одного цвета, запускается рекурсивная цепная реакция с прогрессивным множителем очков (Combo X2, X3 и т.д.).
- Тактическая смена цвета: Игрок видит не только текущий патрон в стволе, но и следующий в очереди, и может мгновенно менять их местами по тапу на орудие для построения более выгодных комбинаций.

# Компоненты игры:

- - Шары цепочки (Path Balls) — движутся по сложной математической траектории сплайна Catmull-Rom. Обладают физикой «паровозика»: задние шары толкают передние вперед, не позволяя элементам накладываться друг на друга.
- - Орудие-стрелок (Shooter) — центральная турель, плавно вращающаяся на 360 градусов вслед за пальцем игрока (прицеливание по тапу/удержанию), отвечающая за расчет вектора запуска снарядов.
- - Снаряды в полете (Fired Balls) — высокоскоростные объекты, летящие по заданной траектории. В реальном времени просчитывают коллизии (столкновения) и обеспечивают бесшовную вставку нового шара в нужный индекс движущегося поезда.
